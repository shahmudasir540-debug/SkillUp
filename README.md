# SkillUp 🚀

**SkillUp** is a premium, AI-powered career acceleration platform designed to bridge the gap between where you are and where you want to be. 

By analyzing your resume and career goals, SkillUp generates personalized, 6-month learning roadmaps, provides smart gap analysis, and simulates job fit for target roles.

## ✨ Key Features
- **📄 AI Resume Parser**: Effortlessly extract skills and experience from PDF or LinkedIn JSON.
- **🗺️ Personalized Roadmaps**: Dynamic, monthly learning paths tailored to your specific dream role.
- **🔬 Smart AI Gap Analysis**: Identify missing technical and soft skills instantly.
- **🎯 Job Fit Simulator**: Compare your profile against any job description.
- **🗓️ Visual Timelines**: Interactive Gantt charts to track your learning progress.
- **💎 SkillUp Pro**: Premium tier with complete roadmap access and advanced AI tools.

## 🛠️ Technology Stack
- **Frontend/Logic**: [Streamlit](https://streamlit.io/)
- **AI Engine**: [Google Gemini 2.0 Flash-Lite](https://ai.google.dev/)
- **Data Viz**: Plotly Express
- **PDF Processing**: reportlab & PyPDF2

## 🚀 Getting Started

### Prerequisites
- Python 3.10+
- Google Gemini API Key

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/shahmudasir540-debug/SkillUp.git
   cd SkillUp
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```
